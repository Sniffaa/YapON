# YapON
Users may create posts, share ideas, ask questions, and chat with others.
### Dependencies
* express
* cors
* dotenv
* jsonwebtoken
* bcryptjs
* multer
* mysql2
* axios
* react-router-dom
* concurrently
### Installing
```git clone https://github.com/Sniffaa/YapON.git```
### Executing
From the root directory use the command ```npm start```
